# Парсинг Telegram-каналов и анализ упоминаний брендов

Инструмент для скачивания постов из публичных Telegram-каналов и анализа упоминаний Яндекса, VK, Сбера и Ozon. На выходе — отчёты с тональностью по каждому автору.

## Установка

```bash
pip install -r requirements.txt
```

Для скачивания постов нужны Telegram API-ключи (`api_id` и `api_hash`) — их можно получить на [my.telegram.org](https://my.telegram.org). При первом запуске скрипт попросит ввести номер телефона и код из Telegram. Дальше авторизация сохраняется и повторный ввод не нужен.

## Как пользоваться

### 1. Скачать посты из каналов

```bash
python3 scrape.py --channels username1 username2 --since 2025-10-01
```

- Вместо `username1` — имя канала без `@` (можно несколько через пробел)
- `--since` — с какой даты качать посты (обязательно)
- `--until` — до какой даты (по умолчанию — сегодня)

Результат сохраняется в папку `exports/` — по одному JSON-файлу на канал.

### 2. Построить отчёт

```bash
python3 analyze_channels.py
```

Скрипт сам находит все скачанные каналы, ищет в них упоминания брендов и создаёт:

- `reports/report.md` — сводный отчёт по всем каналам
- `reports/by_author/` — отдельный файл по каждому автору

При первом запуске тональность постов ещё не определена — в отчётах будет пометка «не размечено».

### 3. Разметить тональность

Тональность размечается в чате Cursor. Попросите агента:

> Разметь тональность по файлу `artifacts/llm_sentiment_queue.jsonl`

Агент прочитает посты, оценит отношение автора к каждому бренду и сохранит результат. Если постов много — результат разбивается на части, и их нужно склеить:

```bash
python3 merge_sentiment_batches.py
```

### 4. Пересобрать отчёт с тональностью

```bash
python3 analyze_channels.py
```

Теперь отчёты содержат оценку тональности для каждого упоминания бренда.

## Что делает каждый файл

| Файл | Что делает |
|------|------------|
| `scrape.py` | Скачивает посты из Telegram-каналов в `exports/` |
| `analyze_channels.py` | Анализирует посты и строит отчёты в `reports/` |
| `content_enrich.py` | Подтягивает текст из YouTube-роликов, если в посте есть ссылка (запускается автоматически) |
| `merge_sentiment_batches.py` | Склеивает батчи разметки тональности в один файл |
| `llm_sentiment_label.py` | Альтернативная разметка тональности через OpenAI API (опционально) |

## Структура папок

```
exports/          — скачанные посты каналов
reports/          — готовые отчёты
  report.md       — сводный
  by_author/      — по каждому автору
artifacts/        — промежуточные файлы (очередь на разметку, батчи)
```
