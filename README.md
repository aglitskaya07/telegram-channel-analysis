# Парсинг Telegram и анализ упоминаний брендов

## Скрипты

| Файл | Назначение |
|------|------------|
| `scrape.py` | Выгрузка постов из публичных Telegram-каналов |
| `analyze_channels.py` | Поиск упоминаний брендов, генерация отчётов |
| `content_enrich.py` | Подтягивание транскриптов YouTube для более точного анализа |
| `merge_sentiment_batches.py` | Склейка результатов разметки тональности |
| `llm_sentiment_label.py` | Опциональная автоматическая разметка через OpenAI API |

## Генерируемые папки

| Папка | Содержимое |
|-------|------------|
| `exports/` | Скачанные посты каналов |
| `reports/` | Готовые отчёты |
| `artifacts/` | Промежуточные файлы для разметки |

## Установка

```bash
pip install -r requirements.txt
```
