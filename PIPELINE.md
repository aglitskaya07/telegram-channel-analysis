# Как пользоваться проектом

## Что здесь лежит

| Файл / папка | Зачем |
|---|---|
| `scrape.py` | Скачивает посты из Telegram-каналов |
| `analyze_channels.py` | Анализирует посты: находит упоминания Яндекса, VK, Сбера и Ozon, строит отчёты |
| `content_enrich.py` | Автоматически подтягивает текст из YouTube-роликов, если в посте есть ссылка (подключается сам, запускать отдельно не нужно) |
| `merge_sentiment_batches.py` | Склеивает результаты разметки тональности в один файл |
| `exports/` | Папка со скачанными постами (по файлу на каждый канал) |
| `reports/` | Папка с готовыми отчётами — сводный и по каждому автору отдельно |
| `artifacts/` | Служебная папка с промежуточными данными |
| `sentiment_overrides.json` | Файл с оценками тональности постов |

## Как скачать посты из канала

```bash
python3 scrape.py --channels username1 username2 --since 2025-10-01
```

- Вместо `username1` — имя канала без `@` (можно несколько через пробел)
- `--since` — с какой даты качать посты

Результат появится в папке `exports/`.

## Как построить отчёт

```bash
python3 analyze_channels.py
```

Скрипт сам найдёт все скачанные каналы, проанализирует посты и создаст:

- `reports/report.md` — сводный отчёт по всем каналам
- `reports/by_author/` — отдельный отчёт по каждому автору

При первом запуске тональность постов ещё не определена — в отчётах будет пометка «не размечено».

## Как разметить тональность

Тональность размечается в чате Cursor — нужно попросить агента:

> Разметь тональность по файлу `artifacts/llm_sentiment_queue.jsonl`

Агент прочитает посты, оценит отношение автора к каждому бренду и сохранит результат. Если постов много — результат разбивается на части, и их нужно склеить:

```bash
python3 merge_sentiment_batches.py
```

После этого запустите анализ ещё раз:

```bash
python3 analyze_channels.py
```

Теперь отчёты будут уже с тональностью.

## Порядок действий коротко

1. Скачать посты → `python3 scrape.py --channels ... --since ...`
2. Запустить анализ → `python3 analyze_channels.py`
3. Разметить тональность в чате Cursor
4. Запустить анализ повторно → `python3 analyze_channels.py`
5. Готовые отчёты — в папке `reports/`
